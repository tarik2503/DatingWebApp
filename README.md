# DatingApp

A dating application built with ASP.NET Core Web API, Angular and Entity Framework Core.

## Tech Stack

- ASP.NET Core 10
- Entity Framework Core
- SQLite
- Angular (Coming Soon)
- JWT Authentication
- AutoMapper
- SignalR

## Features

- User Registration
- Login
- JWT Authentication
- Member Profiles
- Photo Upload
- Likes
- Messaging
- Presence Tracking

## Architecture

API
|
├── Controllers
├── Data
├── DTOs
├── Entities
├── Interfaces
├── Services
├── Helpers
├── Middleware
